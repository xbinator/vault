# 2026-04-20

## Changed
- 为多标签页接入 KeepAlive 缓存生命周期，支持按标签页保留页面实例并在关闭标签时释放缓存。
- 抽离默认布局的路由 KeepAlive 包装组件逻辑，降低多标签页缓存实现的维护心智负担。
- 在多标签页 KeepAlive 设计文档中补充新增页面接入说明，注明普通页面与特殊页面的处理方式。
- 补齐源码编辑器 Markdown 主题常量模块，修复现有主题测试在类型检查阶段找不到模块的问题。
- 增加多标签页 KeepAlive 缓存实现计划，拆分路由缓存 key、标签页状态和编辑器激活态改造步骤。
- 增加多标签页 KeepAlive 缓存方案设计文档，明确由标签页生命周期管理页面缓存。
- 修复 BPromptEditor 删除变量标签时依赖已脱离 DOM 节点恢复光标的问题，避免 Backspace/Delete 在变量边界抛出异常。
- 修复 BPromptEditor 在零尺寸选区探测菜单位置时使用已移除临时节点恢复选区的问题，改为按父节点偏移恢复折叠光标。
- 修复 BPromptEditor 受控更新时仅按首个文本节点恢复光标的问题，改为记录并恢复编辑器内部节点路径。
- 修复 BPromptEditor 清空内容后 placeholder 不显示的问题，改为基于显式空态标记而不是 `:empty` 判断。
- 修复 BPromptEditor 空内容时将 placeholder 渲染在可编辑内容流内导致光标显示在占位文本后的问题，改为独立覆盖层显示占位提示。
- 修复 BPromptEditor placeholder 显示状态依赖原生 DOM dataset 导致 Vue 不响应更新的问题，改为由编辑器核心维护响应式空态。
- 为 BPromptEditor 增加内部历史栈，支持 `Ctrl/Cmd + Z` 撤销与 `Ctrl/Cmd + Shift + Z` / `Ctrl/Cmd + Y` 重做自定义编辑操作。
- 为 BPromptEditor 新增回归测试，覆盖空态占位与关键交互修复约束。
- 为顶部标签栏增加拖拽排序能力，支持按目标标签左右半区插入到前后位置。
- 为标签页状态管理增加重排持久化测试，确保拖拽后的顺序会写回本地缓存。
- 优化顶部标签栏拖拽体验，增加边缘自动横向滚动与拖拽结束后的误点击保护。
- 修复拖拽最后一个标签到前两个标签之间时，插入指示在左右两侧来回跳动的问题。
