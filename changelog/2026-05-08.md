# 2026-05-08

## Changed
- 将聊天侧边栏压缩消息改为时间线式上下文边界节点，不再向用户展示 `summaryText`，只保留状态与简短说明
- `/compact` 手动压缩接入聊天发送态，支持通过现有停止按钮中止；中止后压缩消息会持久化标记为 `已取消`
- 将 `handleCompactContext` 从 `index.vue` 抽离到 `hooks/useCompactContext.ts`，收敛手动压缩消息的创建、回填与提示反馈逻辑

## Fixed
- 修复旧版 SQLite 数据库升级后 `chat_session_summaries` 缺失 `token_count_snapshot` 等摘要扩展列的问题，避免压缩摘要写入时报 `table chat_session_summaries has no column named ...`
- 修复 `/compact` 压缩任务中止时 `index.vue` 未透传取消回调，导致停止按钮已复位但压缩消息未立即切换为 `已取消`` 的问题

## Tests
