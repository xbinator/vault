# 2026-03-29

## Added
- 新增笔记自动保存功能，支持桌面端和 Web 端
  - 桌面端：自动写入文件系统
  - Web 端：自动存储到 IndexedDB（使用 localForage）
  - 新建文件不自动保存，需用户手动保存后开启
  - 支持配置自动保存开关和延迟时间（默认 2 秒防抖）
- 新增 `useAutoSave` Hook，提供自动保存状态管理和立即保存功能
- 新增 `indexedDBStorage` 工具，封装 IndexedDB 文件存储操作
- 新增编辑器头部保存状态 UI 反馈（保存中/已保存/未保存/保存失败）

## Changed
- 修复 BEditor 在页面存在多个实例时目录锚点和滚动定位可能串台的问题。
- 修复 BEditor 的 `editable` 属性在运行时切换后未同步到编辑器实例的问题。
- 将 BEditor 的编辑器扩展、内容同步和目录锚点逻辑拆分到 `hooks` 目录中的 `useExtensions`、`useContent`、`useAnchors`，降低组件复杂度。
- 修复 BEditor 中无序列表项点击时光标容易跳到下一行的问题，收紧列表项段落间距以保证焦点落点稳定。
- 修复 BEditor 无序列表在点击时可能触发 `Called contentMatchAt on a node with invalid content` 的问题，规范列表项 Markdown 解析结果，确保始终生成合法的 `paragraph block*` 结构。
- 扩展 `settings` store，新增 `autoSaveEnabled` 和 `autoSaveDelay` 配置项
- 扩展 `Native` 接口，新增 `autoSave` 方法
- 完善 `useFileActive` hook，实现新建、打开、保存、另存为完整功能
- `useFileActive` hook 新增标题同步功能，编辑器标题输入框失焦时自动更新窗口标题
- 重构编辑器视图，集成 `useAutoSave` hook 并同步 `editorStore` 状态
- 为 BEditor 增加块级悬浮提示，鼠标移入正文显示 `T`，移入标题显示对应 `H1-H6` 标识。
- 将 BEditor 的悬浮提示封装为独立的 `HoverIndicator` 组件。
