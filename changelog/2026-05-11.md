# 2026-05-11

## Changed
- 新增“编辑器设置重构”设计文档，明确编辑器独立偏好存储、视图设置迁移与真实写盘保存策略方案。
- 新增“编辑器设置重构”实现计划文档，拆分编辑器偏好迁移、保存策略执行层与设置页接入的开发步骤。
- 重构编辑器设置存储，新增独立的 `editorPreferences` 用于管理默认视图模式、大纲显示、页宽和保存策略。
- 新增设置中心“编辑器”页面，统一配置编辑器视图偏好与真实写盘保存策略。
- 为编辑器新增 `manual`、`onBlur`、`onChange` 三种真实磁盘保存策略，并保持未落盘文档只写应用草稿。
- 优化聊天侧边栏语音输入的实时转写体验，将增量文本同步写入输入框而不只显示在波形区域。
- 修正语音分段并发转写时的乱序显示问题，仅展示按顺序连续完成的实时文本。
- 修复语音会话重置后旧异步转写结果可能污染新录音会话的问题。
- 补充聊天侧边栏语音输入的集成回归测试，覆盖 `voice-start`、`voice-partial` 与 `voice-complete` 的同范围替换链路。
- 补充空转写结果的父组件集成回归测试，确保语音占位会被清空并提示错误。
- 调整 `write_file` 与 `edit_file` 工具：当目标路径命中当前激活文档时，优先通过编辑器替换文档内容，而不是直接写磁盘文件。
- 补充 `write_file` 与 `edit_file` 的工具回归测试，覆盖命中当前激活文档时走 `replaceDocument` 的分支。
- 恢复 `read_file` 对 `documentId` 的输入兼容：优先通过编辑器上下文解析当前文档路径，缺失时再回退到显式 `path`。
- 修复 `read_file` 在缺少 `path` 时直接抛出运行时异常的问题，并补齐 `documentId` 相关回归测试。
- 修复富文本编辑器在整篇内容替换后仍使用旧选区范围重算浮层位置的问题，失效选区会被及时清理，避免 `coordsAtPos` 与 `textBetween` 越界。
- 修复 `CurrentBlockMenu` 在文档更新后继续消费旧悬停块位置的问题，越界位置会被安全丢弃并收起菜单。
- 补充 `useSelectionAssistant` 与 `CurrentBlockMenu` 的 stale position 回归测试，覆盖文档变化后的失效位置清理路径。
- 抽离 `BSettingsPage` 公共布局组件，统一编辑器、语音组件、模型服务三个设置页的根容器 + Header + Body 布局。

## Features
- 为 `BPromptEditor` 增加光标位置读取与指定范围文本替换能力，用于承载语音实时占位更新。
