# 2026-04-22

## Added
- 新增 `BChat` 模型消息转换缓存能力，支持复用未变化的历史消息前缀。
- 新增聊天消息 `parts` 结构化片段，支持文本、思考、工具调用与工具结果的有序展示和持久化。
- 新增 `BChat` 结构化消息片段设计文档，记录 `parts` 写入、渲染、模型转换和持久化约定。

## Changed
- 拆分 `BChat` 服务配置读取与缺失配置时的 UI 提示逻辑，避免 `getServiceConfig` 混合业务和副作用。
- 提取 `resetToolLoopState()`，将请求中止时的运行时状态清理与新会话初始化解耦。
- 为 `BChat` 消息转换新增缓存回归测试，覆盖追加消息复用与历史消息变更失效两种场景。
- 调整 `BChat` 消息渲染与模型上下文转换逻辑，界面优先展示结构化 `parts`，`content` 保留为纯文本聚合内容。
- 为 `BChat` 深度思考片段补充折叠交互，并在工具调用入参为空时隐藏空白参数代码块。
- 调整 `BChat` 工具续轮的完成事件时机，确保中间轮次的工具调用和工具结果也会持久化到聊天记录。
- 为 `BChat` 工具结果片段补充折叠交互，并让工具续轮继续复用当前 assistant 消息，避免工具结果后拆成新的消息气泡。
- 调整 `MessageBubble` 直接基于 `BBubble` 渲染，移除对 `BBubbleText` 的容器层依赖，收紧聊天气泡组件职责边界。

## Removed
- 无

## Features
- 优化长消息历史下的模型消息构建开销，减少工具续轮时重复转换。
