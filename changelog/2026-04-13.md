# 2026-04-13

## Features
- 在 `BChat/index.vue` 中引入 `Messages` 组件并结合 `Container.vue` 展示消息流，补充消息发送、编辑、复制、删除、重新生成等基础交互逻辑。
- 新增 `chokidar` 依赖用于监听本地文件的外部修改与删除事件。
- 支持当文件在外部修改时，弹窗提示用户重新加载新内容或丢弃修改。
- 支持当文件在外部删除时，弹窗提示用户，将文件重置为未命名状态并保留当前编辑内容。
- macOS 下新增原生菜单栏，并与编辑器工具栏功能保持一致。

## Changed
- 使用 `@vueuse/core` 中的 `useStorage` 封装编辑器的辅助工具侧边栏显隐状态 (`visible`) 以及宽度 (`width`)，将其统一为 `sidebarState` 进行缓存。
- 完善主进程 `FileWatchService` 模块监听与事件派发。
- 更新 `ElectronAPI` 以及 Native 抽象层的接口定义，增加 `watchFile`, `unwatchFile`, `onFileChanged` 接口。
- 修改 `useFileActive` 的 Hook 逻辑，挂载文件变化与删除事件监听。
- 修改辅助侧边栏，未配置大模型时弹窗提示并引导前往设置页。
- 扩展 `Modal.confirm` 的选项，支持自定义确认和取消按钮的文本。
- macOS 下隐藏编辑器工具栏，改由系统菜单触发相同业务逻辑，并同步主题/视图的选中状态到菜单。
- 新增 `Messages.vue` 组件，使用 `BBubbleText` 统一渲染左右聊天气泡，支持气泡内显示操作栏（复制、编辑、删除、重新生成）。

## Removed
- 移除了冗余的 `MessageList.vue` 和 `MessageItem.vue` 组件。
