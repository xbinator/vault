## Why

当前编辑器的 Markdown 样式比较基础，缺少 Typora 那样的美观阅读体验。Typora 以其简洁、优雅的 Markdown 编辑体验闻名，仿照其样式可以大幅提升用户体验。

## What Changes

- 优化 Markdown 渲染样式，接近 Typora 风格
- 调整标题、段落、列表、代码块等样式
- 添加更好的间距和排版
- 支持代码高亮主题

## Capabilities

### New Capabilities
- typora-styling: 仿照 Typora 的 Markdown 样式

### Modified Capabilities
- (无)

## Impact

- 更好的 Markdown 编辑体验
- 更美观的阅读体验
- 代码块支持语法高亮
