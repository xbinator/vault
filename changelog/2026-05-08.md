# 2026-05-08

## Added
- 新增 BModelSelect 全局组件，支持以模态对话框形式选择模型
- 支持搜索过滤模型
- 支持按提供商分组展示模型列表
- 新增 `useHeaderTabsPragmaticDrag.ts` 组合模块，基于 Pragmatic Drag and Drop 封装标签页拖拽排序

## Changed
- HeaderTabs 标签页拖拽排序从原生 HTML Drag and Drop 迁移到 Pragmatic Drag and Drop
- 收缩 `headerTabDrag.ts`，移除被库替代的纯计算函数，保留类型定义与 `closestEdgeToMovePosition` 转换函数
- `HeaderTabs.vue` 退化为薄组件，拖拽逻辑委托给 `useHeaderTabsPragmaticDrag` 模块

## Features
- `/model` 斜杠命令现在打开 BModelSelect 模态对话框
- BModelSelect 组件支持 v-model:open 和 v-model:model 双向绑定
- BModelSelect 组件支持程序化打开（ref.open()）
