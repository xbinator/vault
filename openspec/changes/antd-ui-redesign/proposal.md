## Why

当前编辑器使用原生 HTML 样式，缺少专业的 UI 组件体验。使用 ant-design-vue 组件可以快速实现美观、一致的界面，提供更好的用户体验。

## What Changes

- 使用 ant-design-vue 的 Layout 组件重构整体布局
- 使用 Button、Dropdown 组件重构工具栏
- 使用 Input、TextArea 组件优化输入体验
- 添加 Ant Design 主题支持（亮色/暗色）

## Capabilities

### New Capabilities
- antd-integration: 使用 ant-design-vue 组件重构 UI

### Modified Capabilities
- (无)

## Impact

- UI 组件化：使用 Ant Design 组件
- 主题支持：原生支持亮色/暗色主题
- 代码结构：组件更加模块化
