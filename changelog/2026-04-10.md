# 2026-04-10

## Changed
- 将 Ant Design 全局 `message` 组件配置收敛到 `plugins`：限制最大显示数量为 1，顶部距离为 100px，持续时间为 3 秒。
- 修复 `SelectionAIInput`（AI 划词面板）触发时可能导致页面意外滚动到上一次选区的问题。
- 修复 `ai-selection-highlight` 高度与原生选中高亮不一致的问题，通过设置垂直 `padding` 及 `box-decoration-break` 属性使二者对齐。
- 优化 `BScrollbar` 和 `BTruncateText` 的尺寸监听方式，降低 `ResizeObserver loop completed with undelivered notifications` 警告出现概率。
- 将 TypeScript 调整为最新稳定版 `5.9.2`，移除对 TypeScript 6 兼容参数的依赖。
- 将 Electron 主进程准备脚本改为跨平台 Node 脚本，修复 Windows 下 `npm run electron:dev` 无法直接启动的问题。
- 隐藏 Electron 窗口标题栏：macOS 使用 `titleBarStyle: 'hidden'`，Windows/Linux 使用 `frame: false`。
- 将 Electron 主进程和 preload 脚本迁移到 ESM，移除 CommonJS 兼容 hack（`electron/package.json` 和 `prepare-electron-main.mjs`）。
- 修复 ESM 迁移后 `electron-store` 加密密钥不稳定导致无法读取存储数据的问题，改为将密钥持久化到独立文件。
- 修复 ESM 模式下 `__dirname` 未定义的问题，使用 `import.meta.url` 和 `fileURLToPath` 替代。
- 统一窗口标题管理：将 `setWindowTitle` 调用集中到 `setting store`，组件通过 store 获取标题状态。

## Added
- 新增 `BTitleBar` 组件，为 Windows/Linux 提供自定义标题栏，包含最小化、最大化、关闭按钮。
- 新增窗口控制 IPC 接口：`window:minimize`、`window:maximize`、`window:close`、`window:isMaximized`。
- `setting store` 新增窗口标题管理：`setWindowTitle`、`resetWindowTitle`、`initWindowTitle` 方法，支持标题持久化。

## Refactored
- `BTitleBar` 组件标题改为从 `setting store` 获取，移除 `title` prop。
- `useFileActive` 使用 `settingStore.setWindowTitle` 替代直接调用 `native.setWindowTitle`。
- 路由后置守卫使用 `settingStore.setWindowTitle` 替代直接调用 `native.setWindowTitle`。
- `App.vue` 启动时调用 `settingStore.initWindowTitle()` 恢复上次标题。

## Documentation
- 为 Electron 主进程文件添加详细注释：
  - `electron/main/index.mts` - 主进程入口、IPC 处理器
  - `electron/main/database.mts` - SQLite 数据库管理
  - `electron/main/store.mts` - 安全存储管理
  - `electron/main/migration.mts` - Tauri 数据迁移
  - `electron/preload/index.mts` - 预加载脚本 API 暴露
