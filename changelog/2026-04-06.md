# 2026-04-06

## Added
- 新增 ModelModal 组件，支持创建和编辑 AI 模型
- 为 ModelList 组件添加创建和编辑模型的功能
- ModelModal 组件支持配置模型的基本信息和能力配置

## Changed
- 修改 ModelList 组件的 "创建 AI 模型" 按钮点击事件，打开 ModelModal 对话框
- 为 ModelList 中的每个模型项添加编辑按钮

## Features
- ModelModal 支持配置模型的最大上下文窗口
- ModelModal 支持配置模型的视觉识别、深度思考、联网搜索、图片生成和视频识别能力
- ModelModal 支持选择模型类型（对话、图片、视频、多模态）

## Changed
- 更新 ProviderModel 接口定义，使其与 ModelModal 中的模型配置一致
- 统一使用 ProviderModel 类型替代 Model 类型，确保类型定义的一致性
- ModelModal 组件的模型类型选择器改为使用 options 方式配置
- ModelModal 组件的最大上下文窗口输入改为使用 ASlider 滑动输入条
- ModelModal 组件的滑动条标记点配置改为使用常量定义
- ModelModal 组件移除了模板中未使用的属性，简化了代码结构
- DetailHeader 组件内容合并到 detail.vue 文件中，移除了单独的 DetailHeader 组件
- ProviderHeader 组件内容合并到 index.vue 文件中，移除了单独的 ProviderHeader 组件
- detail.vue 中的编辑按钮样式修改为与 editor/index.vue 中的设置按钮样式一致
