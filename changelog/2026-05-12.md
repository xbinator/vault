# 2026-05-12

## Changed
- 新增“WebView 双实现并行”设计文档，明确 `shared`/`native`/`web` 目录拆分、双路由入口、共享状态模型与 `<webview>` 宿主安全策略。
- 根据设计审查意见细化“WebView 双实现并行”文档，补充 keep-alive 生命周期边界、`will-attach-webview` 拒绝反馈链路、受控 `partition` 策略与加载进度语义。
- 新增“WebView 双实现并行”实现计划文档，拆分共享层抽离、`native` 迁移、宿主安全控制、`web` 页面接入与双路由验证步骤。
- 拆分 WebView 共享层、`native` 实现与 `<webview>` 标签实现，并新增 `/webview/native` 与 `/webview/web` 两个显式入口。
- 为 `<webview>` 模式补充主进程 `webviewTag` 宿主安全策略、独立分区和 attach-rejected 反馈链路。
- 收紧 HeaderTabs 右键菜单弹出行为，限制同一时刻仅允许一个菜单打开，并要求等待前一个关闭动画完成后再打开下一个。
- 新增“HeaderTabs 右键关闭菜单与批量关闭”设计文档，明确 tabs store 的关闭计划/执行分层、五类关闭动作规则、脏标签确认条件与激活回退策略。
- 新增“HeaderTabs 右键关闭菜单与批量关闭”实现计划文档，拆分 store 规则测试、关闭计划实现、HeaderTabs 右键菜单接入、组件验证与 changelog 回归步骤。
- 新增 HeaderTabs 右键关闭菜单，并把关闭、关闭其他、关闭右侧、关闭已保存、全部关闭统一收敛到 tabs store 的关闭计划/执行接口。
- 新增“表格 NodeView 增删控件”设计文档，明确富文本表格的自定义 `Table NodeView` 结构、分割线新增与行列删除入口、最外边框插入规则、命中优先级与命令执行边界。
- 新增“表格 NodeView 增删控件”实现计划文档，拆分表格几何命中、命令映射、`TableView` NodeView 接入、样式迁移与交互测试步骤。
- 为富文本表格新增自定义 `Table NodeView`，支持分割线新增行列、整行整列删除入口、最外边框插入规则与滚动后的 hover 重算。
- 调整 `TableView` overlay 结构为滚动容器外渲染，修复 `overflow-x: auto` 导致的分割线高亮与新增按钮被裁剪问题。
- 收敛表格控件布局：列新增/删除按钮固定在顶部，行新增/删除按钮固定在左侧，不再在表格中部显示入口。
- 调整表格删除入口为双控件同时显示：悬浮单元格内容区时，顶部显示删列按钮，左侧显示删行按钮。
- 修复表格 hover 控件提前消失问题：鼠标从表格内容区移向顶部/左侧按钮时，不再因为 `scroller` 的 `mouseleave` 提前清空 hover 状态。
- 优化表格增删按钮视觉：接入 `@iconify/vue` 图标，改为跟随编辑器主题的小圆点样式，移除固定绿色和厚重浮层外观。
- 调整表格按钮定位为中心锚点：新增按钮居中贴合分割线，删列按钮在顶部水平居中，删行按钮在左侧垂直居中。
- 调整表格按钮绘制区域：为顶部和左侧预留 gutter，让新增/删除按钮视觉上悬挂在表格容器外侧，同时保留安全可见区避免被裁剪。
- 收敛表格外侧按钮布局：移除 `viewport` 的 `padding-top` / `padding-left`，恢复表格本体原始布局，仅通过绝对定位让按钮半悬挂在容器外侧。
- 调整表格按钮最终定位：顶部按钮上偏移半个按钮尺寸，左侧按钮左偏移半个按钮尺寸，在不改变表格布局的前提下实现更准确的外侧悬挂效果。
