# 2026-04-17

## Changed

- 调整 `BPromptEditor` 的默认回车行为为直接发送，使用 `Ctrl+Enter` 或 `Cmd+Enter` 插入换行，并移除回车调试日志输出。
- 完善 `BChat` 消息重新生成流程，新增 `setSessionMessages` 用于同步会话消息持久化状态。
- `BSearchRecent` 移除文件时同步移除对应的标签页。
- 调整 `tabsStore` 的 `addTab` 更新逻辑，在同一标签已存在时覆盖标题与路径，确保编辑器保存后解析出的文件名能同步刷新标签页标题。
- 为 `BChatSidebar` 增加聊天空状态缺省图，未开始对话时展示引导插画和说明文字。
- 优化 `BEditor` 的 Markdown round-trip 行为，在富文本模式未发生语义变更时尽量保留用户原始输入中的 HTML 注释和引用式链接写法，并让 HTML 注释、引用式链接及其定义在富文本视图中以可编辑文本显示。
- 为 `BEditor` 富文本模式补充点击链接时使用系统浏览器打开外链的能力。
- 优化 `setting.ts` 中的 `normalizeSettings` 函数，使用 `lodash-es` 的 `defaultsDeep` 函数简化代码结构，同时保持必要的类型检查。
- 清理 `BEditor` 目录侧栏相关组件中的未使用样式，移除未命中的空状态与目录占位类定义。
- 调整 `BEditor` 目录侧栏标题归属，将顶部标题点击区域从 `AnchorContent` 抽离到 `Sidebar` 统一管理。
- 为 `BEditor` 侧栏标题区增加文件图标与更多菜单，支持重命名、删除和打开文件所在位置，并补充对应的文件操作链路。
- 将 `BEditor` 侧栏菜单中的“打开 Finder 位置”文案调整为跨平台的“打开所在位置”。
- 同步应用菜单中的视图与主题勾选状态，确保启动时和切换后“源代码模式”等菜单项与设置状态一致。
- 修复编辑器删除文件后最近文件历史与标签页残留的问题，删除时会一并清理同一路径关联的本地记录与标签页。
- 修复 macOS 上无法使用 Cmd+C/Cmd+V 复制粘贴的问题，在 Electron 菜单配置中补充标准的剪切、复制、粘贴、全选菜单项。
- 修复系统级快捷键闪烁问题，在全局 keydown 监听器中排除系统级编辑快捷键（复制、粘贴、剪切、全选），避免干扰系统默认行为。
- 修复 Windows/Linux 平台菜单被设置为 null 导致系统级编辑快捷键失效的问题，现在所有平台都有完整的菜单配置。
- 调整 Electron 应用菜单中的剪切、复制、粘贴、全选为显式转发到当前聚焦窗口，并在 Electron 环境下取消前端对保存快捷键的重复注册，减少保存和粘贴时的系统菜单闪动。
