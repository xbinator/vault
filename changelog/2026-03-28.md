# 2026-03-28

## Added
- 为 BEditor Heading 扩展添加 ID 属性支持，标题渲染时自动生成唯一 ID
- 为 BEditor 添加滚动监听，滚动时自动高亮侧边栏对应目录项

## Changed
- 将 UnoCSS 配置中的 `presetUno()` 替换为 `presetWind3()`，消除弃用提示并保持原有原子类能力
- 将 Toolbar 文件菜单命令分发改为策略映射，替代 switch 分支判断
- 将编辑器页的 Toolbar 交互改为父组件传入菜单项并统一回收单一 action 事件，减少模板级碎片化绑定
- 将编辑器页文件菜单进一步收敛为带 `onClick` 的配置项，消除独立的命令分发表和事件处理函数
- 将编辑器页文件菜单配置抽离到 `src/views/editor/hooks`，使页面组件仅保留编辑器状态与文件读写流程
- 为 `DropdownOption` 类型字段补充语义注释，便于复用和维护
- 将 `src/utils/file` 重命名为 `src/utils/native`，同步统一原生能力抽象与实现类命名，避免模块职责被误解为仅处理文件
- 为 BEditor 补充 TipTap placeholder 对应的空内容样式，修复正文为空时占位文案不显示的问题
- BEditor 侧边栏点击目录项时平滑滚动到对应标题位置
