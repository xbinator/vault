# 2026-04-18

## Added
- 为 BEditor 源码模式接入 CodeMirror 6，提供 Markdown 编辑、高亮、行号和历史记录能力。
- 新增 BEditor 富文本主角组件 `RichEditorHost`，承接 Tiptap、Front Matter 和富文本搜索逻辑。

## Changed
- 重构 BEditor 编辑器适配层，将源码编辑器和富文本编辑器统一到 `EditorController` 控制接口。
- 精简 BEditor 入口组件，使 `index.vue` 专注于布局、模式编排、目录和查找栏转发。
- 新增 `docs/ai-tools` 文档目录，记录 AI Tools 设计与实施计划
- 新增 AI Tools 实施计划，拆分类型、编辑器上下文、内置工具、Electron 流式事件、聊天接入和验证任务
- 新增 AI Tools 设计记录，明确内置读写编辑工具 MVP、renderer 编排方案及未来 custom/MCP 扩展边界
- 优化模型服务配置卡片布局，Prompt 改为列表项展示并通过编辑弹窗修改
- 调整模型服务 Prompt 编辑弹窗为确认式提交，点击确定后才保存修改
- 抽离服务商设置页 `SidebarSection`、`SidebarItem`、`SidebarSearch` 组件，降低布局组件复杂度并修正自定义服务商折叠显示逻辑
- 新增 `useAutoCollapse` Hook，服务商设置页支持容器宽度小于 800px 时自动折叠侧边栏
- 设置页外层侧边栏接入 `useAutoCollapse`，容器宽度小于 800px 时自动折叠菜单
- 将设置页外层菜单和服务商侧边栏折叠状态保存到 setting store
- 将 Mermaid 预览能力合并到代码块组件，统一 `mermaid` 语言代码块的源码、预览、复制和折叠交互
- 修复空代码块切换到 Mermaid 后无法输入的问题，空内容时默认保持源码编辑态
- 优化 Mermaid 代码块输入体验，避免输入首个字符后自动切换到预览态
- 修复 Mermaid 代码块在预览和编辑模式切换后源码内容不可见、无法编辑的问题
- 支持在代码块内连续按 `Ctrl/Cmd + A` 时，第一次选中当前代码块内容，第二次全选全文
- 重构 storage 模块，抽离公共函数到 utils 目录
  - 创建 `utils/database.ts`：统一数据库访问封装（dbSelect, dbExecute, isDatabaseAvailable）
  - 创建 `utils/json.ts`：统一 JSON 序列化工具（parseJson, stringifyJson, parseJsonArray）
  - 重构 `chats/sqlite.ts`：使用公共函数替换重复代码
  - 重构 `providers/sqlite.ts`：使用公共函数替换重复代码
  - 重构 `service-models/sqlite.ts`：使用公共函数替换重复代码

## Features
- Mermaid fenced code 现在通过普通代码块渲染，并可直接在代码块内切换图表预览
- 提升代码可维护性：减少约 150+ 行重复代码
- 统一错误处理：JSON 解析、数据库访问统一处理
- 提高类型安全：统一的类型定义和泛型支持
