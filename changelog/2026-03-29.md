# 2026-03-29

## Added
- 新增笔记自动保存功能，支持桌面端和 Web 端
  - 桌面端：自动写入文件系统
  - Web 端：自动存储到 IndexedDB（使用 localForage）
  - 新建文件不自动保存，需用户手动保存后开启
  - 支持配置自动保存开关和延迟时间（默认 2 秒防抖）
- 新增 `useAutoSave` Hook，提供自动保存状态管理和立即保存功能
- 新增 `indexedDBStorage` 工具，封装 IndexedDB 文件存储操作

## Changed
- 修复 BEditor 在页面存在多个实例时目录锚点和滚动定位可能串台的问题。
- 修复 BEditor 的 `editable` 属性在运行时切换后未同步到编辑器实例的问题。
- 将 BEditor 的编辑器扩展、内容同步和目录锚点逻辑拆分到 `hooks` 目录中的 `useExtensions`、`useContent`、`useAnchors`，降低组件复杂度。
- 扩展 `settings` store，新增 `autoSaveEnabled` 和 `autoSaveDelay` 配置项
- 扩展 `Native` 接口，新增 `autoSave` 方法
