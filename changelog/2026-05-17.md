# 2026-05-17

## Added
- 新增 MCP 工具请求配置类型、聊天请求透传和主进程工具过滤边界。
- 新增 MCP 设置页路由与基础配置页面。
- 新增 MCP 工具到 AI SDK 工具集的 server-scoped 装配能力。
- 新增 MCP 本地 stdio runner、运行时状态、discovery cache 与 IPC/preload 通道。
- 新增模型可调用的 MCP 配置读写工具，支持读取配置、新增/更新/删除 server 以及刷新 discovery。
- 新增 `BJsonGraph` JSON 可视化编辑器、节点图主题、自定义节点组件，以及 JSON 源码解析/布局/联动 hooks。
- 新增 JSON 编辑器相关测试、`json-source-map` 类型声明和 `EditorDriver` 注册表。

## Changed
- 扩展工具设置 Store，支持 MCP server 的增删改和持久化归一化。
- AI 服务统一追加 MCP 工具说明词到系统提示。
- MCP 设置页显示本地执行/discovery 状态，并支持手动刷新 discovery。
- AI 服务会将 discovery cache 中通过 allowlist 过滤的 MCP 工具注册为 AI SDK 工具，并通过本地 runtime 执行。
- 更新 MCP 工具接入设计文档，移除云端 sandbox 和默认调用配置范围，改为本地 stdio 执行方案。
- 修复编辑器查找条在富文本与源码模式切换后无法直接继续回车跳转搜索结果的问题。
- 补充 JSON 图编辑器设计文档的可维护性方案，引入编辑器驱动注册表以避免 editor 页面按文件类型持续膨胀。
- MCP 设置页将添加/编辑弹窗抽离为独立的 `ServerEditorModal`，并改用 CodeMirror JSON 编辑器承载配置输入。
- MCP 设置页补全 server 运行状态展示，并修正编辑保存时仅更新可编辑字段，保留启用状态和连接超时等 server 级配置。
- 为 MCP 设置页补充页面级、编辑器 helper 与 `ServerEditorModal` 组件级测试，覆盖状态展示、字段保留、JSON 回填与校验阻断。
- 编辑器页面改为通过 `EditorDriver` 动态解析 Markdown/JSON 驱动，并将 AI 上下文注册统一收口到页面层。
- `BEditor` 补齐 `EditorController` 暴露方法，`useBindings` 和 `useFileSelection` 同步兼容新的统一编辑器协议。
